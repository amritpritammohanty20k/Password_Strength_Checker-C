# Password_Strength_Checker-C
A simple C program that checks the strength of a password, validates it using multiple security rules, and optionally saves strong passwords to a file for future reference.
# Features
  1. Check password strength
  2. Generate a strong password
  3. Save a new password
  4. View all saved passwords
  5. Search password by label
  6. Update a password
  7. Delete a password
  8. Delete all passwords
  9. Exit
# Project Structure
Password_Strength_Checker-C/
│
├── Password_Checker.c        # Main C program
├── saved_passwords.txt       # File where strong passwords get stored
└── README.md                 # Project documentation
# About saved_passwords.txt
This file stores the strong passwords generated or validated by the program.
It is included in this repository so teachers/reviewers can see the sample output.
